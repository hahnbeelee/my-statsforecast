### mstl

> ``` text
>  mstl (x:numpy.ndarray, period:Union[int,List[int]],
>        blambda:Optional[float]=None, iterate:int=1,
>        s_window:Optional[numpy.ndarray]=None,
>        stl_kwargs:Optional[Dict]={})
> ```

|            | **Type**                                | **Default** | **Details**          |
|------------|-----------------------------------------|-------------|----------------------|
| x          | ndarray                                 |             | time series          |
| period     | typing.Union\[int, typing.List\[int\]\] |             | season length        |
| blambda    | typing.Optional\[float\]                | None        | box-cox transform    |
| iterate    | int                                     | 1           | number of iterations |
| s_window   | typing.Optional\[numpy.ndarray\]        | None        | seasonal window      |
| stl_kwargs | typing.Optional\[typing.Dict\]          | {}          |                      |
